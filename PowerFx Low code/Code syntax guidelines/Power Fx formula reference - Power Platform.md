Detailed summary

- The Microsoft Power Fx formula reference provides information about Power Fx formulas that can be used within the Power Platform, which includes various components such as Canvas apps, Dataverse, Model-driven apps, Power Automate, Power Pages, and Power Platform CLI.
- The Power Fx formulas are available for specific sub-areas of these components, including Dataverse formula columns and functions, and Power Automate desktop flows, while they are not applicable to Canvas apps, Model-driven apps, Power Pages, and Power Platform CLI.
- The Power Fx formula reference information is organized in a way that allows users to filter by Power Platform component, with each component listed as a filter article in the table of contents, providing a list of all available formulas for that component.
- The sub-areas for each formula reference article are listed under the "Applies to" area, indicating the specific components and areas where the formula can be used, such as Canvas apps, Model-driven apps, and Dataverse formula columns.
- Additional resources are available, including formula reference articles for Model-driven apps, Dataverse formula columns, and Dataverse plug-ins, which can be accessed through the "See also" section.
- The page also provides options for users to provide feedback, suggest fixes, and access additional resources, including the Ask Learn AI assistant, which can answer questions and clarify concepts using trusted Microsoft documentation.
- The Microsoft Power Fx formula reference page was last updated on 09/04/2025, and users can sign in to access the Ask Learn feature and other resources.
