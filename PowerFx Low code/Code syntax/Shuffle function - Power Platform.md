Concise summary

- The Shuffle function randomly reorders the records of a table in Power Platform.
- It returns a table with the same columns and number of rows as the original table.
- The syntax for the Shuffle function is `Shuffle( Table )`, where Table is the required table to be shuffled.


Detailed summary

- The Shuffle function in Power Platform is a feature that randomly reorders the records of a table, and it applies to various platforms, including Canvas apps, Copilot Studio, Desktop flows, Model-driven apps, Power Platform CLI, and Dataverse functions.
- The Shuffle function returns a table that has the same columns and number of rows as the argument table, effectively creating a rearranged version of the original table without altering its structure or content.
- The syntax of the Shuffle function is straightforward, requiring only one argument, which is the table to be shuffled, denoted as Shuffle(Table) where Table is the required input.
- An example of using the Shuffle function is demonstrated with a collection named Deck, which contains details about playing cards, and applying the formula Shuffle(Deck) would return a copy of the Deck collection that has been randomly shuffled.
