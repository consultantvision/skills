Concise summary

- The Mod function in Power Platform returns the remainder of a division.
- It takes two required parameters: Number (the number to divide) and Divisor (the number to divide by).
- The result has the same sign as the Divisor.


Detailed summary

- The Mod function in the Power Platform is a mathematical operation that returns the remainder of a division, and it applies to various platforms, including Canvas apps, Copilot Studio, Desktop flows, Dataverse formula columns, Model-driven apps, Power Platform CLI, Dataverse functions, and Power Pages.
- The Mod function takes two required parameters: Number, which is the number to divide, and Divisor, which is the number to divide by, and the result has the same sign as the divisor.
- The syntax of the Mod function is Mod(Number, Divisor), where both the Number and Divisor are required inputs, and the function returns the remainder after the division operation is performed.
- The Mod function can handle both positive and negative numbers, and the result will have the same sign as the divisor, as demonstrated in the examples, such as Mod(25, 4) returning 1, Mod(25, -4) returning -3, Mod(-25, -4) returning -1, and Mod(-25, 4) returning 3.
